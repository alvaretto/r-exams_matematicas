El Problema

Imagina que tienes tres cestas de frutas, y cada una tiene una combinación diferente de manzanas, plátanos y naranjas. Sabemos cuántas piezas de cada fruta hay en cada cesta y cuánto cuesta cada cesta en total, pero no sabemos cuánto cuesta cada fruta individualmente. Queremos encontrar el precio de cada fruta para poder calcular el precio de una cuarta cesta.

Las Ecuaciones

Podemos escribir tres ecuaciones usando las tres cestas de frutas. Supongamos que xxx es el precio de una manzana, yyy es el precio de un plátano, y zzz es el precio de una naranja. Las ecuaciones serían algo así:

  1. ax+by+cz=p1ax + by + cz = p_1ax+by+cz=p1​
  2. dx+ey+fz=p2dx + ey + fz = p_2dx+ey+fz=p2​
  3. gx+hy+iz=p3gx + hy + iz = p_3gx+hy+iz=p3​

Donde a,b,c,d,e,f,g,h,ia, b, c, d, e, f, g, h, ia,b,c,d,e,f,g,h,i son las cantidades de cada fruta en las cestas, y p1,p2,p3p_1, p_2, p_3p1​,p2​,p3​ son los precios totales de las cestas.

Método de Eliminación

Para resolver este sistema de ecuaciones, podemos usar el método de eliminación. Aquí están los pasos:

  1. Elegir dos ecuaciones y eliminar una variable: Tomamos dos ecuaciones y las combinamos de tal manera que una de las variables desaparezca. Por ejemplo, podemos multiplicar la primera ecuación por un número y luego restarla de la segunda ecuación para eliminar xxx.

  2. Repetir el paso 1 con otras ecuaciones: Hacemos lo mismo con otras dos ecuaciones para eliminar la misma variable. Ahora tenemos dos nuevas ecuaciones con solo dos variables.

  3. Resolver la nueva ecuación: Usamos las dos nuevas ecuaciones para encontrar una de las variables restantes, como yyy.

  4. Encontrar la otra variable: Sustituimos el valor encontrado en una de las ecuaciones originales para encontrar la otra variable, como zzz.

  5. Encontrar la variable eliminada: Finalmente, sustituimos los valores encontrados en una de las ecuaciones originales para encontrar la variable que eliminamos inicialmente, como xxx.

  6. Calcular la cuarta cesta: Usamos los precios de las frutas que encontramos para calcular el precio de la cuarta cesta.

Conclusión

Siguiendo estos pasos, podemos encontrar los precios de las manzanas, plátanos y naranjas, y luego calcular el precio de la cuarta cesta de frutas, que es 483.
